# MIRAI
### or simply Mechanistic Interpretablity and Reversing of AI

#### The What?
This project aims to create a simple and user-friendly application, that allows users to do reverse engineering and analyis on any pre trained pytorch model layer by layer. 

#### The Why?
One the major problems of AI is the inexplicability of their results, and the blackbox nature of the executions.

this project aims to allow users to look at the inner wiring and unravel the exact equations that a pre-trained NN has learned.
